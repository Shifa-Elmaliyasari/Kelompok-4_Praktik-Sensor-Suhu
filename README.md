# Tugas 1 - Implementasi Teks Berjalan pada OLED SSD1306
## Deskripsi
Program ini menampilkan teks berjalan pada layar OLED SSD1306 menggunakan Arduino.
## Library yang digunakan
- Adafuit SSD1306
- aDAFRUIT GFX
## Cara Kerja
Teks akan bergerak dari kanan ke kiri secara terus menerus.
## Output
Tampilan teks berjalan pada layar OLED
